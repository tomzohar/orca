# API Layer

This directory contains the presentation layer of the `agent-jobs` module.

## Responsibilities

- Handling HTTP requests and responses.
- Defining DTOs (Data Transfer Objects) for input validation.
- Orchestrating SSE (Server-Sent Events) for real-time job updates.

## Key Files

- `agent-jobs.controller.ts`: Main entry point for API requests.
- `dto/`: Request validation schemas.
